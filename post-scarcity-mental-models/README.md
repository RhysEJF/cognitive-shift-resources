# Post-Scarcity Mental Models

Mental models extracted from Charles Stross's *Accelerando* using the `extract-mental-models` skill. These models help analyze decisions through a post-scarcity lens - scenarios where attention, reputation, and compute have flipped what's scarce.

## Installation

1. Copy all `.md` files to your `memory/frameworks/post-scarcity/` directory
2. Copy the `chains/` directory to `memory/frameworks/post-scarcity/chains/`
3. Install the extraction skill from `../skills/extract-mental-models/` to your `skills/` directory

## Usage

### Quick Start
```
/post-scarcity "your decision here"
```

### With Deep Context
```
/post-scarcity "your decision here" --deep
```

## Default Chain

1. **Substrate Independence of Cognition** → Surface "human-only" assumptions
2. **Agalmic Economy** → Reframe value flow (exchange vs gift/reputation)
3. **Abundance Graduation** → Check if good should exit market entirely
4. **Turing-Complete Regulatory Automation** → Map rules onto code vs human discretion
5. **First-Case Precedent Lock-In** → Name precedent today's choice sets

## Example Use Cases

- **Financial planning** in AI-augmented careers
- **Product strategy** for near-zero marginal cost goods
- **Regulatory compliance** where rules become code
- **Market analysis** for reputation economies
- **Investment decisions** in post-scarcity sectors
