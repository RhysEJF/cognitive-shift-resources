# Cognitive Shift Resources

Open frameworks and skills from [The Cognitive Shift](https://thecognitiveshift.com).

## Skills

### [SWARMS Assessment](skills/swarms-assessment.md)
A structured scoring framework for evaluating ideas against six dimensions of distributed intelligence: Sense, Wage, Adapt, Replicate, Mobilize, Sustain. Based on [How to Win with Agentic Swarms: A Tactical Field Guide](https://leanpub.com/agentic-swarms-tactical-field-guide/).

Use it to audit your AI system, rank improvement ideas, or assess strategic alignment with swarm intelligence principles.

### [Field Guide Factory](skills/field-guide-factory.md)
A proven methodology for transforming any topic into a publication-ready tactical field guide using AI agent swarms. Codified from two successful books: *How to Win with Agentic Swarms* (54K words, 86 tasks) and *The First 90 Days: A New Dad's Field Guide* (414KB, 152 tasks).

Covers the full 5-wave pipeline: Foundation, Research, Synthesis, Writing, Assembly.

### [Analyze Repo](skills/analyze-repo.md)
A structured skill for evaluating external repositories, protocols, or tools against your own system's architecture. Produces integration briefs that surface overlaps, borrowable ideas ranked by impact, and what to avoid.

Use it to build a research library during roadmap planning.

## How to Use These Skills

These are markdown files designed to be injected into AI agent context. You can:
- Feed them directly to Claude, ChatGPT, or any LLM as system instructions
- Include them in your agent orchestration framework as skill files
- Use them as reference methodologies for your own workflows
